My macOS/iOS setup (with config) is described [here](/config).

### Websites

- [React](https://react.dev) to render, [Jazz](https://jazz.tools)/[Reatom](https://v1000.reatom.dev) for state
- Deployed on [Vercel](https://vercel.com)
- Also use [Blade](https://blade.im) with its ORM
- [Panda CSS](https://panda-css.com) or [Tailwind](https://tailwindcss.com) for styling
- [Better Auth](https://better-auth.com) for auth

### HTTP

- TS: [Hono](https://hono.dev) (distributed with Cloudflare) or [Encore](https://encore.dev) (single origin on GCP) depending on task
- Go: [Encore](https://encore.dev)
- Rust: [Axum](https://github.com/tokio-rs/axum)

### CLIs

- Go with [go-snap](https://github.com/dzonerzy/go-snap)
- TS with [Bun](https://bun.sh) & [Optique](https://github.com/dahlia/optique)
- Rust with [clap](https://github.com/clap-rs/clap)

### iOS

- [Expo](https://expo.dev) + Expo plugins
- [Unistyles](https://unistyl.es) for styles
- [Jazz](https://jazz.tools)/[Reatom](https://v1000.reatom.dev) for state
- [Better Auth](https://better-auth.com) for auth

### Desktop

- Wrap websites with [Tauri](https://tauri.app) for desktop apps

### Infra

- Manage services with [Alchemy](https://alchemy.run)

### Analytics

- [Openpanel](https://github.com/Openpanel-dev/openpanel) for analytics (site visits etc.)
- [Sentry](https://sentry.io) for errors
- [Axiom](https://axiom.co) for logs
